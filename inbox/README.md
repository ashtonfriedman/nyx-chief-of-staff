# Inbox

Quick-capture landing zone for unprocessed inputs. Everything here is temporary —
triage it to the right location, then remove it.

## Triage Workflow

For each item, decide:
1. **Move** — route to the correct domain, initiative, or expertise folder
2. **Task** — create an action item in the relevant initiative's `next-actions.md`
3. **Archive** — not actionable, but worth preserving in `Archive/`
4. **Delete** — noise, outdated, or already captured elsewhere

## Conventions

- Don't let items sit here for more than a week
- If you're unsure where something goes, check `mind-index.md` for similar topics
- Inbox zero is the goal — this folder should usually be empty
